## FAQs

| Key | English (US) | Español (MX) |
|-----|-------------|--------------|
| faq.title | FAQs | |
| faq.group_basics | Basics | |
| faq.group_scoring | Scoring & Classifications | |
| faq.group_rules | Rules & Tiebreakers | |
| faq.group_technical | Technical | |
| faq.q_every_match | Do I have to predict every match? | |
| faq.a_every_match | No. You can skip any match. But every correct pick earns points, so the more you predict the better your chances on the leaderboard. | |
| faq.q_exact_required | Is the exact score required? | |
| faq.a_exact_required | No — it's optional. Pick the winner (or draw) first. If you also nail the exact score you earn a +3 bonus on top. | |
| faq.q_change_picks | Can I change my picks? | |
| faq.a_change_picks | Yes, as many times as you like before the day locks. Once a matchday locks (10 min before the first kickoff that day), all picks for that day are final. | |
| faq.q_when_join | When can I join? | |
| faq.a_when_join | You can join any time before the entry deadline (3 days after kickoff). Late joiners can still pick unlocked matches but auto-forfeit anything already locked. | |
| faq.q_points_per_match | How many points can I earn per match? | |
| faq.a_points_per_match | Group stage: up to 5 pts (2 for correct outcome + 3 exact score bonus). Knockout: up to 6 pts (2 outcome + 3 score + 1 for correct advancing team). | |
| faq.q_classifications | What are the different leaderboard classifications? | |
| faq.a_classifications | **Overall** — cumulative points across all matches. **Format** — survivor elimination that mirrors the World Cup structure. **Full Bracket** & **KO Bracket** — pre-tournament bracket predictions scored after groups / knockouts complete. | |
| faq.q_draw_knockout | What does "Draw" mean in a knockout match? | |
| faq.a_draw_knockout | It means the scores are level after extra time — the match is decided by penalties. You pick who advances separately. Your result prediction is the score after extra time, excluding penalties. | |
| faq.q_extra_ko_point | How does the extra knockout point work? | |
| faq.a_extra_ko_point | In knockout matches you also predict which team advances. This is worth 1 bonus point on top of the result and score picks. It matters most when the match is a draw — you can still earn a point by correctly picking who goes through on penalties. | |
| faq.q_tiebreakers | How are ties on the leaderboard broken? | |
| faq.a_tiebreakers | Head-to-head goal difference → head-to-head goals → overall goal difference → overall goals → coin flip. | |
| faq.q_postponed | What happens if a match is postponed? | |
| faq.a_postponed | Your picks stay valid. The match moves to its new time and your predictions carry over. If a match is cancelled, affected picks are voided (no points gained or lost). | |
| faq.q_auto_save | Do my picks auto-save? | |
| faq.a_auto_save | Yes. Winner picks save instantly on tap. Score predictions save after 1.75 seconds of inactivity or when you tap away. | |
| faq.q_desktop | Can I play on desktop? | |
| faq.a_desktop | Yes — the app works on any device with a browser. It's designed mobile-first but fully functional on desktop. | |
